BURIED TREASURE

Time to use your new coding skills to dig up some Treasure!

You are currently at position (0,0) and the treasure is at (150, -100). Move the character to the treasure by using the
moveLeft( ),
moveRight( ),
moveUp( ),
and moveDown( ) functions.
Each moves the character 50px in the indicated direction. Be quick. You only have 3 seconds to reach the destination and each move takes 0.5s.

The coordinates shown are in the x-y plane (x,y). You start off at 0 in the x-direction and 0 in the y-direction. You must get to 150 in the x-direction and -100 in the y-direction. Each move takes you 50 pixels, so divide the total amount you need to move in each direction by 50 to find out how many times to call each function.
