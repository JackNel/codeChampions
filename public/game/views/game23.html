ARRAY DUNGEON

You are searching the Array Dungeon for treasure.

Dungeon doors are numbered like array indices. Your treasure map tells you that you need to open the second and fifth doors.

Use the openDoor()function with the index of the door you want to open as a parameter (i.e. openDoor(0) to open door number 0) to search the right rooms. Be careful, because opening the wrong one will lead you straight to a dungeon monster!
stuff

Remember, array's begin at index 0, so the first item is array[0] and the third item would be array[2]. Also, the openDoor() function only takes a single paramter, so make sure you call it the correct number of times.
