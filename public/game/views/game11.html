BURIED TREASURE Part 1

Let's learn how to use the code editor and play a game!

You are currently at position (0,0) and the treasure is at (0, -50). The moveDown() function which has been entered for you will move your character down by 50 pixels to the treasure.

Hit the 'Run' button above the game and gather your first treasure.
