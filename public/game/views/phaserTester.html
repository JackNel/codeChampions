If/else if Statements

If statements will run the code inside of them if a given condition is true. When an else if statement is present, the code inside of it will run when the condition in the if portion evaluates to false and the conditional inside of the else if statement evaluates to true. An else statement will then run if none of the other conditions were met.


if(hasShip){
fireLaser();
}
else if(friendHasShip){
borrowShip();
}
else{
buyNewShip();
}
