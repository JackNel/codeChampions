GOALS

If the Space Avenger is in position to shoot the alien, fire the laser.

If not, then put the Space Avenger in position and attack depending on if the avenger needs to move to the left or to the right.

To do this we will create an if/elseif/else statement:
if(scanForEnemy()) {
  fireLaser();
} else if {
  attackRight();
} else {
  attackLeft();
}
stuff

Like in the previous lesson, the "scanForEnemy" function will scan to see if the Space Avenger is in range to destroy the alien. Depending on the location of the alien, the Space Avenger will need to attack to the right or to the left. Make sure you're spelling your functions correctly!
