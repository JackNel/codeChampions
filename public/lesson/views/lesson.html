Introduction

Welcome, future Code Champion, to the exciting world of programming. Before we can embark on our adventure, there are some things you should know.

We will be working in a language called JavaScript that is featured on many websites that you visit daily like Google, Facebook, Disney, and even this site!

Like any language, JavaScript has rules. As we progress through the courses, you'll learn more of the rules, but there are a few you should know to start.

  • JavaScript likes to be told when you're done with a certain action. That's why we end lines of code with a semi-colon (;) to tell it to move on to the next thing. Think of it like putting a period at the end of a sentence when writing normally. Failure to put a semi-colon or putting the wrong punctuation can make your code go crazy and not work properly.
  • Spelling and capitalization matter. If you want access to something you need to make sure it is spelled correctly and that all letters are properly capitalized based on how it was defined. If you incorrectly try to access something, you will get an error saying it is 'undefined'.
