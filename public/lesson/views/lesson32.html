Good job! Now we can use the else statement to specify a block of code to be executed if the condition is false.

Example:

If the hour is less than 18, create a "Good day" greeting, otherwise "Good evening":

var hour = 15;
if (hour < 18) {
  greeting = "Good day";
} else {
  greeting = "Good evening";
}

The result of greeting will be:

Good day
