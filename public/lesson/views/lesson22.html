Arrays

An Array in Javascript is a special type of variable that can hold more than one piece of data. Say you have a list of animals: cat, dog, horse, and snake. You can put these all into an array called animals that you can reference as a <collection or you can reference each piece of data inside of the array.

You can create an array the same way you would create a regular variable. The only difference is you will have multiple primitives, surrounded by brackets and separated by commas. Let’s look at an example:

var animals = [“cat”, “dog”, “horse”, “snake”];

As you can see, we now have one variable called animals, that contains four pieces of data. There are many things we can do with an array, but we will only look at a few right now.

Every array has an index. The index automatically assigns a number to each piece of data inside the array. However, arrays are tricky because they use something called a zero-based index. This means that the first number in the index is not 1, it is 0. So, the index for our animals array would look like this:

“cat” = index 0
“dog” = index 1
“horse” = index 2
“snake” = index 3

We can use the index to reference pieces of data inside of the array. For example, we can create a new variable from our array:

var newDog = animals[1];

result:
var newDog = “dog”

Another thing we can do with our array is check how many pieces of data it contains by measuring it’s length. We can make a number variable by measuring the length of the array:

var length = animals.length()

result:
var length = 4

This is another example of how the zero-based index is tricky! Remember, our index range is 0-3, but our index length is 4.

One of the coolest things about arrays are that they can contain different types of data. So, you can have an array that has a string and a number. For example:

var myBirthday = [“September”, 15, 1992];

In this example, “September” is a string and 15 and 1992 are both numbers.
