In Lesson 3, we are going to discover if/else statements.

Sometimes JavaScript requires the ability to make distinctions between different possibilities. This is when an if/else statement will come in handy.

The general syntax for if statements is:

if (condition) {action1} else {action2};

Part 1 will cover basic if statments. Use the if statement to specify a block of JavaScript code to be executed if a condition is true.

Example

Make a "Good day" greeting if the hour is less than 18:00:

var hour = 15;
if (hour < 18) {
  greeting = "Good day";
}

The result of greeting will be:

Good day
