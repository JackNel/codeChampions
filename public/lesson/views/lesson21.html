Variables & Strings

Just like in algebra, JavaScript uses variables. You may have seen in math class something like:

x = 2
y = 5

Solve:
x+ y

or

x + 5 = 10

Solve for x.


(But don't be afraid if you haven't seen that in math class yet)

In Javascript, variables can be thought of as containers that hold data. When creating a variable, you must use the keyword var. The var keyword is followed by an identifier, which is a name created by the coder to reference the specific variable. Identifiers can be simple placeholders, like the above algebra example, or they can be more descriptive for greater code clarity.

Once you create an identifier, the next step is assign a piece of data to the variable. This is done using the = sign, followed by a primitive data type. There are three types of primitives in Javascript:

Strings - A string is snippet of text that can contain letters, numbers, and symbols. A string is defined by surrounding the text in double quotes:

ex1: “This is a String!”
ex2: “I was born in 1992!”

Numbers - A number is any value between negative infinity and positive infinity. A number can also be a decimal.

ex1: 100
ex2: 8.5
ex3: -20

Boolean - A boolean can only be true or false and is used in situations where you need to tell the program if something is true or false! You will learn more about how to use these in lesson 3.

Let’s take a look at some variable declarations:

var myName = “Jack”
var myBirthday = “Sept. 15, 1992”

var x = 5
var myAge = 10

var doorIsClosed = true
var doorIsOpen = false
